# 学生照片文件夹

## 使用方法

1. 把学生照片放到这个文件夹
2. 照片文件名要和 `class-config.json` 中学生的 `photo` 字段一致

## 示例

如果 class-config.json 中写的是：
```json
{ "name": "LI WEI SHEN (李玮绅)", "group": "A", "photo": "li_weisen.jpg" }
```

那照片就放在：
```
photos/li_weisen.jpg
```

## 照片建议
- JPG 或 PNG 格式
- 建议尺寸: 300×400 像素（头像比例）
- 文件大小: 每张不超过 500KB 比较好（加载快）
